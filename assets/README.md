# 📸 MailPilot Assets

This folder contains screenshots and demo videos for the MailPilot project.

## 📁 Folder Structure

```
assets/
├── screenshots/          # Project screenshots
│   ├── dashboard.png     # Main dashboard view
│   ├── email-list.png    # Email list with cards
│   ├── email-read.png    # Email reading view
│   └── ai-response.png   # AI response example
│
└── demo/                 # Demo videos
    └── demo-video.mp4    # Full feature demo
```

## 📸 Screenshot Guidelines

### Recommended Size:
- **Width**: 1920px (or 1280px)
- **Format**: PNG or WebP
- **Quality**: High quality, clear text

### What to Capture:

1. **dashboard.png** - Main dashboard showing:
   - Email list sidebar
   - Chat interface
   - User info

2. **email-list.png** - Email cards showing:
   - Sender, subject, date
   - Read, Archive, Star buttons
   - Nice card design

3. **email-read.png** - Full email view showing:
   - Formatted email content
   - Decoded HTML entities
   - Clickable links

4. **ai-response.png** - AI interaction showing:
   - User command
   - AI response
   - Success message

## 🎬 Video Guidelines

### Demo Video:
- **Length**: 30-60 seconds
- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 or 1280x720
- **Frame rate**: 30fps

### What to Show:
1. Sign in with Google (2-3 sec)
2. Dashboard view (2 sec)
3. Type: "show me emails" (3 sec)
4. Show email list (3 sec)
5. Type: "send email to test@example.com saying Hello" (5 sec)
6. Show success message (2 sec)
7. Type: "summarize my inbox" (5 sec)
8. Show AI summary (3 sec)

## 📝 Tips

- **Use test data** - Don't expose personal emails
- **Blur sensitive info** - Names, email addresses
- **Good lighting** - Clear, readable text
- **Consistent theme** - Same browser, same time of day
- **Add captions** - Brief descriptions help
